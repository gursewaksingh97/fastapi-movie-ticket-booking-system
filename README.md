# 🎬 Movie Ticket Booking Management System

A basic backend CRUD project developed using FastAPI, PostgreSQL, SQLAlchemy ORM and Pydantic Models.

This project allows users to manage movie ticket booking records digitally through API endpoints using Swagger UI and Postman.

## Features

* Add booking records
* View booking details
* Update booking information
* Delete booking records
* PostgreSQL database integration
* Pydantic data validation

## Tech Stack

* FastAPI
* PostgreSQL
* SQLAlchemy ORM
* Pydantic
* Uvicorn

## Workflow

User → Swagger/Postman → FastAPI → SQLAlchemy ORM → PostgreSQL Database

## Project Type

Beginner Backend CRUD API Project
